Computer Glossary

  • CPU: The brain of the computer that processes instructions.
  • RAM: Temporary memory that helps the computer run programs.
  • Hard Drive: Where all your files and data are stored.
  • Browser: A program that lets you visit websites (e.g., Chrome, Firefox).
  • Operating System: Software that manages the computer (e.g., Windows, macOS).
